# Awesome TUI

[![Awesome](https://awesome.re)](https://awesome.re)

> Browse awesome lists in the terminal

A beautiful TUI application for exploring curated awesome lists directly from your terminal. Built with Textual.

## Features

- Browse popular awesome lists from GitHub
- Search through lists
- Beautiful dark theme (Feynman-inspired)
- Fast async loading
- Keyboard shortcuts for navigation

## Installation

```bash
pip install awesome-tui
```

Or install from source:

```bash
git clone https://github.com/12britz/awesome-tui.git
cd awesome-tui
pip install -e .
```

## Usage

```bash
awesome
```

## Controls

| Key | Action |
|-----|--------|
| `↑` / `↓` | Navigate |
| `Enter` | Select list |
| `o` | Open URL in browser |
| `/` | Focus search |
| `r` | Refresh |
| `q` | Quit |
